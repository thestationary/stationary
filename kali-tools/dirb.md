## dirb


```
$ dirb <url_base> [<wordlist_file(s)>] [options]
```


for more info:
```
$ man dirb      or  $ dirb
```
```
$ dirb <website> -r -o dirb.dirb
```

<website> : http://<ip>
-r: is used to prevent recursions
results will be stored in tmp.dirb

-X  <extensions>                -X .php,.html                 for php and html files
